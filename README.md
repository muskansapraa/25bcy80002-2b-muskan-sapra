# React Product Filter

## Project Title  
**Dynamic Product Filter using React**

##  Description  
This project demonstrates how to build a dynamic product filtering interface using React.  
Users can filter products based on categories using a dropdown menu. The UI updates instantly without page reloads, showcasing React state management and responsive design.

---

## Objectives  

- Implement filterable product data  
- Create a sortable dropdown UI  
- Display products using responsive cards  
- Animate layout transitions  
- Demonstrate React state management  

---

## Technologies Used  

- React (JavaScript)   
- CSS3  
- CSS 

---

## Hardware / Software Requirements  

- Windows / Linux / macOS  
- Node.js (v14 or above)  
- npm or yarn  
- VS Code (or any code editor)  
- Modern Web Browser
